## 🎯 部署命令汇总

### 本地运行

```bash
cd iot-teaching-platform
npm install
npm run dev
# 访问 http://localhost:3000
```

### Vercel部署

```bash
# 方式1：CLI
vercel

# 方式2：网站导入GitHub仓库
# https://vercel.com
```

### 华为云OBS

```bash
npm run build
# 上传 out/ 目录到OBS桶
# 配置静态网站托管
```

### Docker

```bash
docker build -t iot-teaching:v1.0 .
docker run -d -p 80:80 iot-teaching:v1.0
```

## 📝 使用建议

### 教师使用流程

1. **课前**：使用备课中心生成教案
2. **课中**：投屏展示知识点、演示模拟器
3. **课后**：查看测验结果、学生数据

### 学生使用流程

1. **课前**：预习教材内容
2. **课中**：听讲、操作模拟器、完成测验
3. **课后**：复习错题、再次练习

### 部署建议

- **学校机房**：本地运行或内网部署
- **公网访问**：华为云OBS（成本最低）
- **快速体验**：Vercel免费部署
