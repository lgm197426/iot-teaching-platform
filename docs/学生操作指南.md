## 三、实践模拟

### 3.1 进入模拟器

**步骤1**：点击左侧"🔬 实践模拟"

**步骤2**：看到4个模拟器

```
┌──────────────────────┐
│  🌡️ 传感器仿真器      │
│  🎴 RFID模拟器        │
│  📨 MQTT仿真器        │
│  🔐 物联门锁模拟器    │
└──────────────────────┘
```

**步骤3**：点击要使用的模拟器

### 3.2 传感器模拟器

**学习目标**：理解传感器如何采集数据

**操作步骤**：

**第1步**：点击"🌡️ 传感器仿真器"

**第2步**：认识三种传感器
```
温度传感器：显示当前温度（°C）
湿度传感器：显示空气湿度（%）
光照传感器：显示光照强度（Lux）
```

**第3步**：点击"▶️ 运行"按钮
- 观察数据开始变化
- 看到曲线图实时绘制

**第4步**：观察数据曲线
```
曲线向上 = 数值增加
曲线向下 = 数值减少
波动是正常的（真实环境也是这样）
```

**第5步**：设置参数
- 采样间隔：多久采集一次数据（1秒/2秒/5秒）
- 温度阈值：温度超过这个值会报警

**第6步**：导出数据（可选）
- 点击"📥 导出CSV"
- 得到一个表格文件
- 可以用来写实验报告

**思考题**：
1. 为什么数据会波动？
2. 采样间隔设多大合适？
3. 什么情况需要设置阈值报警？

### 3.3 RFID模拟器

**学习目标**：理解RFID刷卡原理

**操作步骤**：

**第1步**：点击"🎴 RFID模拟器"

**第2步**：认识界面
```
左侧：卡片列表
中间：门锁状态
右侧：刷卡记录
```

**第3步**：选择一张卡片

**第4步**：点击"刷卡"按钮
- 看到"正在读取卡片..."动画
- 等待1秒，门锁打开！

**第5步**：观察门锁状态
```
🔒 门已锁定 → 刷卡 → 🚪 门已打开
3秒后自动关闭
```

**第6步**：查看刷卡记录
- 右侧显示每次刷卡的时间
- 显示是允许还是拒绝

**第7步**：试试未授权的卡片
- 添加新卡片
- 设置为"未授权"
- 再刷卡，看看什么结果

**思考题**：
1. 有源标签和无源标签有什么区别？
2. 为什么要设置卡片授权？
3. RFID比普通门钥匙好在哪？

### 3.4 MQTT模拟器

**学习目标**：理解发布订阅通信

**操作步骤**：

**第1步**：点击"📨 MQTT仿真器"

**第2步**：理解概念
```
发布者：发送消息的人
订阅者：接收消息的人
Topic：消息的主题（像电视频道）
Broker：消息中转站（像邮局）
```

**第3步**：订阅一个Topic
- 在"订阅Topic"输入框输入：home/temperature
- 点击"📥 订阅"

**第4步**：发布一条消息
- Topic输入：home/temperature
- 消息输入：{"temp": 26}
- 点击"📤 发布消息"

**第5步**：观察结果
- 上方显示你发布的消息
- 下方"收到的消息"也显示了
- 因为你订阅了这个Topic！

**第6步**：试试通配符
- 订阅：home/#
- 发布到：home/humidity
- 也能收到！（#匹配所有子主题）

**思考题**：
1. 发布订阅和微信聊天有什么不同？
2. QoS 0、1、2分别是什么意思？
3. Topic命名有什么规则？

### 3.5 智能门锁模拟器

**学习目标**：了解多种开锁方式

**操作步骤**：

**第1步**：点击"🔐 物联门锁模拟器"

**第2步**：认识5种开锁方式

**方式1：密码开锁**
1. 点击"🔢 密码开锁"
2. 看到虚拟键盘
3. 输入密码：123456
4. 点击"✓"确认
5. 门锁打开！

**方式2：指纹开锁**
1. 点击"👆 指纹开锁"
2. 系统识别指纹
3. 门锁打开！

**方式3：刷卡开锁**
1. 点击"💳 刷卡开锁"
2. 模拟刷卡
3. 门锁打开！

**方式4：远程开锁**
1. 点击"📱 远程开锁"
2. 模拟手机APP控制
3. 门锁打开！

**方式5：人脸识别**
1. 点击"👤 人脸识别"
2. 看到识别框动画
3. 选择识别的用户
4. 识别成功，门锁打开！

**第3步**：查看开锁记录
- 下方显示每次开锁的时间、方式、用户

**第4步**：注意电量
- 右上角显示门锁电量
- 每次开锁消耗一点电量

**思考题**：
1. 5种开锁方式哪种最安全？
2. 为什么要有开锁记录？
3. 电量耗尽怎么办？

## 四、测验答题

### 4.1 进入测验

**步骤1**：点击"✏️ 测验中心"

**步骤2**：选择要测验的课程
```
第1单元 走进物联网
  └─ 第1课 认识物联网 [3题]
```

**步骤3**：点击课程，开始答题

### 4.2 答题操作

**看到题目后**：

**【单选题】**
```
题目：物联网的英文缩写是什么？
  ○ A. IoT
  ○ B. IT
  ○ C. IoP
  ○ D. NET
```
- 点击你认为正确的选项
- 选中的会变成蓝色

**【多选题】**
```
题目：以下哪些是物联网的特征？（多选）
  □ A. 全面感知
  □ B. 可靠传输
  □ C. 高速计算
  □ D. 智能处理
```
- 可以选多个答案
- 点击选择/取消选择

**【判断题】**
```
题目：物联网就是互联网。
  ○ A. 正确
  ○ B. 错误
```
- 选择正确或错误

**【填空题】**
```
题目：物联网三层架构从下到上依次是：
感知层、____层、应用层。
```
- 在输入框输入答案
- 注意不要写错别字

### 4.3 答题技巧

**技巧1：先看题目要求**
- 是单选还是多选？
- 题目问的是什么？

**技巧2：不确定的先跳过**
- 点击"下一题"先做后面的
- 最后再回来做

**技巧3：填空题检查**
- 注意单位（°C、%、秒等）
- 不要有错别字

**技巧4：善用题号跳转**
```
[1] [2] [3] [4] [5]
```
- 点击题号可以快速跳转
- 蓝色 = 已答
- 灰色 = 未答

### 4.4 提交答卷

**步骤1**：答完所有题目后，点击"提交答卷"

**步骤2**：确认提交
- 确认所有题目都答了
- 点击确认

**步骤3**：查看成绩
```
🎉 测验完成！

得分：8 / 10 分
正确率：80%
```

### 4.5 查看答案解析

**提交后**，会显示每道题的详细情况：

**【答对的题】**
- 显示绿色 ✓
- 简单显示正确

**【答错的题】**
- 显示红色 ✗
- 显示你的答案
- 显示正确答案
- 显示详细解析

```
你的答案：B
正确答案：A
💡 解析：物联网的英文是Internet of Things，
缩写为IoT，其中I代表Internet...
```

**重要**：认真看解析，理解为什么选这个答案！

## 五、错题复习

### 5.1 查看错题本

**步骤1**：点击"✏️ 测验中心"

**步骤2**：看到"❌ 错题本"区域

**步骤3**：点击"展开"查看所有错题

```
错题列表：
1. 第1课 认识物联网 - 错误2次
2. 第3课 物联网架构 - 错误1次
...
```

### 5.2 错题复习方法

**方法1：看解析理解**
- 认真阅读答案解析
- 理解为什么选这个答案
- 不懂就问老师或同学

**方法2：重做错题**
- 找到错题对应的课程
- 重新测验该课程
- 看这次能不能答对

**方法3：标记已掌握**
- 确定已经理解了
- 点击"✓"标记已掌握
- 错题本会移除这道题

### 5.3 错题本的作用

- ✅ 自动记录答错的题
- ✅ 显示错误次数
- ✅ 方便集中复习
- ✅ 考前重点看错题

## 七、常见问题

### Q1：忘记网址怎么办？

**A**：问老师或同学，或者保存到收藏夹。

### Q2：网页打不开怎么办？

**A**：
1. 检查网络是否连接
2. 刷新网页（按F5）
3. 问老师或机房管理员

### Q3：测验可以重做吗？

**A**：可以，点击"重新测验"即可，系统记录最高分。

### Q4：错题本可以删除吗？

**A**：标记"已掌握"后会自动移除，不建议直接删除。

### Q5：模拟器数据可以保存吗？

**A**：点击"导出CSV"保存到电脑，可以用来写实验报告。

### Q6：可以在家里用吗？

**A**：可以，只要有网就能访问。建议用电脑，手机屏幕太小。

### Q7：学习进度会丢失吗？

**A**：不会，进度保存在浏览器中。但换电脑或清除浏览器数据会丢失。

### Q8：如何提高测验成绩？

**A**：
1. 认真学习课程内容
2. 动手操作模拟器理解原理
3. 错题要弄懂，不要死记答案
4. 多复习重点内容

## 附录：6单元学习路线

### 第1单元：走进物联网（5课时）

**学习重点**：
- 物联网的定义和特征
- 物联网的发展历程
- 物联网三层架构
- 物联网的应用场景

**实践建议**：
- 打开首页，浏览物联网应用案例
- 记住物联网三大特征

### 第2单元：感知与识别（6课时）

**学习重点**：
- 传感器原理
- RFID技术
- 条码和二维码
- 图像识别、语音识别

**实践建议**：
- 使用传感器模拟器理解传感器工作
- 使用RFID模拟器理解刷卡原理

### 第3单元：传感器与数据采集（5课时）

**学习重点**：
- 温湿度传感器
- 光敏传感器
- 运动传感器
- 数据采集和处理

**实践建议**：
- 传感器模拟器深入使用
- 尝试设置不同参数观察结果

### 第4单元：物联网通信（6课时）

**学习重点**：
- 蓝牙通信
- Wi-Fi通信
- MQTT协议
- 网关和边缘计算

**实践建议**：
- MQTT模拟器理解发布订阅
- 思考不同通信方式的应用场景

### 第5单元：物联网应用（5课时）

**学习重点**：
- 智能家居
- 智慧农业
- 智慧交通
- 系统集成

**实践建议**：
- 智能门锁模拟器理解智能家居
- 思考生活中的物联网应用

### 第6单元：物联网安全（3课时）

**学习重点**：
- 安全风险
- 数据隐私
- 安全策略

**实践建议**：
- 在门锁模拟器中理解权限控制
- 思考如何保护个人信息
