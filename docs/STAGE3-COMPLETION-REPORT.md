## 🎯 功能特点

### 1. 教材同步学习
- 完整覆盖人教版6单元30课内容
- Markdown渲染，支持代码块、表格
- 每课包含：教学目标、重点、难点、完整内容

### 2. 备课助手
- 智能生成完整教案，节省教师备课时间
- 三种教学风格，适应不同课程特点
- 45分钟课堂流程详细设计

### 3. 物联网模拟器
- 完全基于教材案例设计
- 可视化交互，实时数据展示
- 代码注释通俗，适合八年级学生
- 每个模拟器配有实验指导和思考题

### 4. 测验系统
- 自动判分，即时反馈
- 错题本自动收录，支持复习
- 详细答案解析，帮助理解

## 📝 技术实现

### 前端技术
- Next.js 14 + React 18
- TypeScript 5
- Tailwind CSS 3
- Canvas API（传感器数据曲线）
- LocalStorage（数据持久化）

### 数据存储
- 本地JSON文件（教材、题库、模拟器配置）
- LocalStorage（用户进度、错题本、测验记录）

### 组件设计
- 模块化组件，易于维护
- TypeScript类型定义，代码安全
- 响应式设计，适配不同屏幕

## 🚀 使用说明

### 启动项目

```bash
cd iot-teaching-platform
npm install
npm run dev
```

访问 http://localhost:3000

### 功能入口

| 功能 | 路由 |
|------|------|
| 首页 | `/` |
| 教材学习 | `/course` |
| 课程详情 | `/course/[lessonId]` |
| 备课中心 | `/teaching` |
| 模拟器列表 | `/simulator` |
| 模拟器详情 | `/simulator/[simulatorId]` |
| 测验中心 | `/quiz` |
| 课程测验 | `/quiz?lesson=[lessonId]` |
| 教师后台 | `/teacher` |
