## 一、快速开始

### 1.1 登录系统

**步骤1**：打开浏览器（推荐Chrome或Edge），输入网址

**步骤2**：在右上角切换身份为"教师"

```
┌─────────────────────────────────────┐
│  物联学堂        身份：[学生] [教师] │
└─────────────────────────────────────┘
```

**步骤3**：点击"登录"按钮（演示版本可直接使用）

### 1.2 熟悉界面

**侧边栏导航**：
- 🏠 首页：课程入口和统计
- 📚 教材学习：浏览教材内容
- 📝 备课中心：生成教案（教师专用）
- 🔬 实践模拟：物联网模拟器
- ✏️ 测验中心：测验和成绩
- 👨‍🏫 教师后台：数据管理

## 二、备课中心使用

### 2.1 生成教案（5分钟完成）

**适用场景**：课前准备，生成45分钟标准教案

**操作步骤**：

**步骤1**：点击侧边栏"📝 备课中心"

**步骤2**：在左侧选择要备课的单元和课程
```
第1单元 走进物联网
  └─ 第1课 认识物联网  ← 点击选择
```

**步骤3**：选择教案模板（三选一）

| 模板 | 适用场景 | 特点 |
|------|----------|------|
| 🎤 讲授型 | 知识点较多的课程 | 以教师讲解为主 |
| 🔬 探究型 | 实验类课程 | 引导学生自主探究 |
| 📋 项目型 | 综合应用课程 | 项目式学习 |

**步骤4**：点击"🎯 一键生成教案"按钮

**步骤5**：查看生成的教案内容

生成内容包含：
- ✅ 教学目标（知识与技能、过程与方法、情感态度）
- ✅ 教学重点与难点
- ✅ 教学方法
- ✅ 教学过程（45分钟四个环节）
- ✅ 板书设计
- ✅ 课堂活动设计
- ✅ 课后作业

**步骤6**：保存教案
- 点击"📋 复制"：复制到剪贴板，粘贴到Word
- 点击"📥 下载Markdown"：下载.md文件

### 2.2 教案模板说明

#### 讲授型教案

**时间分配**：
- 导入新课：5分钟
- 讲授新课：20分钟
- 实践操作：15分钟
- 课堂小结：5分钟

**适用课程**：
- 知识点密集的课程
- 需要系统讲解的概念
- 新知识导入

#### 探究型教案

**时间分配**：
- 创设情境：5分钟
- 自主探究：15分钟
- 交流分享：15分钟
- 总结提升：10分钟

**适用课程**：
- 实验类课程
- 可以动手操作的内容
- 培养探究能力

#### 项目型教案

**时间分配**：
- 项目发布：5分钟
- 方案设计：10分钟
- 项目实施：20分钟
- 成果展示：10分钟

**适用课程**：
- 综合应用类课程
- 单元末尾的总结课
- 培养综合能力

## 三、课堂投屏演示

### 3.1 投屏准备

**步骤1**：连接教室投影仪
- HDMI线连接电脑和投影仪
- 设置显示器复制模式
- 调整分辨率为1024×768或1366×768

**步骤2**：打开浏览器，进入网站

**步骤3**：按F11进入全屏模式（可选）

### 3.2 知识点投屏讲解

**步骤1**：点击"📚 教材学习"

**步骤2**：选择要讲解的课程
```
第1单元 走进物联网
  └─ 第1课 认识物联网
```

**步骤3**：全屏展示课程内容
- 滚动展示知识点
- 配合讲解重点难点
- 学生观看投影学习

**技巧**：
- 使用翻页键（空格/方向键）控制进度
- 重要内容可以用浏览器放大（Ctrl + +）
- 光标指向要强调的地方

### 3.3 模拟器投屏演示

**步骤1**：点击"🔬 实践模拟"

**步骤2**：选择要演示的模拟器

**演示传感器**：
1. 进入"传感器仿真器"
2. 设置参数（采样间隔、阈值）
3. 点击"▶️ 运行"
4. 学生观察数据曲线实时变化
5. 设置温度阈值，演示报警功能

**演示RFID**：
1. 进入"RFID模拟器"
2. 点击"刷卡"模拟刷卡
3. 学生观察门锁开闭状态
4. 演示未授权卡片拒绝通行

**演示MQTT**：
1. 进入"MQTT仿真器"
2. 订阅Topic（如home/temperature）
3. 发布消息
4. 学生观察发布订阅机制

**演示智能门锁**：
1. 进入"物联门锁模拟器"
2. 演示5种开锁方式（密码、指纹、刷卡、远程、人脸）
3. 展示开锁日志记录

## 四、模拟器演示

### 4.1 传感器模拟器演示步骤

**教学目标**：让学生理解传感器数据采集过程

**演示流程**（5-10分钟）：

**第1步**：讲解传感器类型
```
"同学们，我们先来认识三种常用传感器：
- 温度传感器：测量环境温度
- 湿度传感器：测量空气湿度
- 光照传感器：测量光照强度"
```

**第2步**：运行仿真
- 点击"▶️ 运行仿真"
- 指向实时数据面板："看，数据在实时变化"

**第3步**：解释数据曲线
```
"这条曲线展示了温度随时间的变化，
可以看到温度在25°C左右波动，
这是正常的环境温度变化"
```

**第4步**：演示阈值报警
- 设置温度阈值为28°C
- 解释："当温度超过28°C时会触发报警"
- 观察报警状态变化

**第5步**：导出数据
```
"我们可以把采集的数据导出保存，
点击'导出CSV'，就得到一个表格文件，
方便后续分析"
```

### 4.2 RFID模拟器演示步骤

**教学目标**：理解RFID工作原理和门禁应用

**演示流程**（5分钟）：

**第1步**：展示RFID标签
```
"这些就是RFID标签，
每个标签有唯一的ID号，
就像我们的身份证一样"
```

**第2步**：演示刷卡
- 选择"学生卡-张明"
- 点击"刷卡"
- 观察读取动画和门锁状态

**第3步**：演示权限控制
```
"注意，不是所有卡片都能开门，
我们来看未授权的卡片"
```
- 添加新卡片，设置为未授权
- 刷卡，观察"拒绝通行"

**第4步**：查看日志
```
"系统会记录每次刷卡，
这就是门禁系统的追溯功能"
```

### 4.3 MQTT模拟器演示步骤

**教学目标**：理解发布订阅通信模式

**演示流程**（5-8分钟）：

**第1步**：概念讲解
```
"MQTT就像一个邮局：
- 发布者把消息投递到邮局
- 订阅者告诉邮局想要什么类型的信
- 邮局把匹配的消息送到订阅者"
```

**第2步**：订阅Topic
- 在订阅框输入"home/temperature"
- 点击"📥 订阅"

**第3步**：发布消息
- Topic：home/temperature
- 消息：{"temp": 26.5}
- 点击"📤 发布"

**第4步**：观察结果
```
"看，订阅了home/temperature的用户
收到了刚才发布的消息"
```

**第5步**：演示通配符
```
"使用#可以订阅所有子主题，
比如订阅home/#，
会收到home/temperature和home/humidity"
```

### 4.4 智能门锁演示步骤

**教学目标**：了解多种开锁方式和工作原理

**演示流程**（5-8分钟）：

**第1步**：展示5种开锁方式
```
"智能门锁支持5种开锁方式：
1. 密码开锁 - 输入数字密码
2. 指纹开锁 - 验证指纹
3. 刷卡开锁 - RFID卡片
4. 远程开锁 - 手机APP控制
5. 人脸识别 - 摄像头识别"
```

**第2步**：演示密码开锁
- 点击"🔢 密码开锁"
- 在键盘输入123456
- 观察门锁打开

**第3步**：演示人脸识别
- 点击"👤 人脸识别"
- 观察识别动画
- 选择识别用户

**第4步**：查看开锁日志
```
"每次开锁都会记录，
包括时间、方式、用户，
这就是安全追溯"
```

## 五、测验管理

### 5.1 发起课堂测验

**步骤1**：点击"✏️ 测验中心"

**步骤2**：选择要测验的课程
```
第1单元 走进物联网
  └─ 第1课 认识物联网 - [5题] ← 点击
```

**步骤3**：学生开始答题
- 告诉学生打开网站，进入测验
- 选择相应课程

**步骤4**：查看实时统计（教师后台）
- 点击"👨‍🏫 教师后台"
- 切换到"测验"标签
- 查看答题进度和正确率

### 5.2 查看测验结果

**步骤1**：进入教师后台

**步骤2**：点击"📊 数据统计"标签

**步骤3**：查看各项数据
- 平均分
- 正确率
- 高频错题
- 知识点掌握情况

**步骤4**：导出数据
- 点击"📥 导出学生成绩"
- 得到Excel文件

### 5.3 分析学生错题

**步骤1**：进入教师后台 → 数据统计

**步骤2**：查看"高频错题"列表
- 显示错误率最高的题目
- 分析学生薄弱知识点

**步骤3**：针对性讲解
- 根据错题分布调整教学重点
- 对薄弱知识点补充讲解

## 七、常见问题

### Q1：教案可以修改吗？

**A**：可以。生成后复制到Word，自由编辑调整。

### Q2：模拟器数据可以保存吗？

**A**：可以。点击"导出CSV"保存数据，实验报告可以保存提交。

### Q3：测验可以多次参加吗？

**A**：可以。学生可以重新测验，系统记录最高分。

### Q4：如何查看学生是否在学习？

**A**：教师后台可以看到学生在线状态和学习进度。

### Q5：可以在手机上使用吗？

**A**：可以，但建议使用电脑，体验更好。投影演示必须用电脑。

### Q6：断网了还能用吗？

**A**：可以。首次加载后，核心功能支持离线使用。

### Q7：如何布置课后作业？

**A**：
1. 教师后台 → 作业管理
2. 选择课程，设置截止时间
3. 发布作业

### Q8：可以自定义题库吗？

**A**：当前版本暂不支持，后续版本会添加。

## 附录：45分钟课堂流程建议

### 推荐流程

**课前5分钟**
- 打开网站，投屏准备
- 学生进入网站

**导入5分钟**
- 投屏展示应用场景图片/视频
- 提问引入主题

**讲解15分钟**
- 投屏展示知识点
- 讲解重点难点
- 学生听讲记录

**演示5分钟**
- 打开模拟器演示
- 学生观看操作

**实践10分钟**
- 学生打开模拟器动手操作
- 教师巡视指导

**测验5分钟**
- 发起课堂测验
- 学生答题
- 查看结果

**小结5分钟**
- 总结知识点
- 布置课后任务
