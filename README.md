# 物联学堂 - 八年级物联网教学平台

> 基于《义务教育信息科技 八年级全一册·物联网实践与探索》的配套教学网站

## 🎯 项目简介

物联学堂是一个专为初中信息科技课堂设计的物联网教学平台，完整覆盖人教版教材6个单元30课时内容，提供教材学习、备课辅助、实验模拟、测验评价等一体化教学解决方案。

### 核心特点

- 📚 **教材同步**：6单元30课完整知识点，对标人教版教材
- 📝 **智能备课**：一键生成45分钟标准教案（3种模板）
- 🔬 **仿真实验**：4种物联网模拟器（传感器、RFID、MQTT、门锁）
- ✏️ **自动测验**：42道测验题，自动判分，错题本
- 🖥️ **课堂适配**：支持1024×768分辨率，低配电脑流畅运行
- 📴 **离线可用**：静态导出，断网也能使用

## 🚀 快速开始

### 环境要求

- Node.js 18.x 或更高版本
- npm 或 pnpm 或 yarn

### 安装与运行

```bash
# 1. 进入项目目录
cd iot-teaching-platform

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 生产构建

```bash
# 构建静态文件
npm run build

# 输出目录：out/
# 可部署到任意静态服务器（Nginx、Apache、GitHub Pages等）
```

### 1. 教材同步学习

| 功能 | 说明 |
|------|------|
| 单元导航 | 6个单元树形导航 |

## 🎨 设计规范

### 颜色

- 主色：教育蓝 `#3b82f6`
- 辅色：白色、浅灰
- 强调：绿色（成功）、红色（危险）

### 字体

- 基础字号：16px
- 最小字号：14px
- 行高：1.6
- 字体：Microsoft YaHei、系统默认

### 按钮

- 最小高度：48px（符合触控标准）
- 最小宽度：96px
- 圆角：12px
- 间距：12px

### 分辨率

- 最小支持：1024×768
- 推荐：1366×768（教室投影仪）
- 最佳：1920×1080

## 📖 使用场景

### 课堂教学（45分钟）

```
课前（5分钟）
├── 教师：使用备课中心生成教案
└── 学生：打开网站，进入教材学习

课中（35分钟）
├── 教师投屏：展示知识点讲解
├── 教师演示：使用模拟器演示
├── 学生实践：动手操作模拟器
└── 课堂测验：发起测验检验效果

课后（5分钟）
├── 教师查看测验统计
└── 学生查看错题，标记重点
```

### 学生自主学习

```
自学流程
├── 选择课程学习知识点
├── 打开模拟器动手实践
├── 完成测验检验学习效果
└── 查看错题本复习巩固
```

### 教师备课

```
备课流程
├── 选择要备课的课程
├── 选择教案模板（讲授/探究/项目）
├── 一键生成完整教案
├── 根据实际情况调整
└── 下载或复制使用
```
